## 🔍 בדיקות מתקדמות

### FIFO & Virtual Lock
- [ ] מלאי זמין יורד מיד כשנוצרת הזמנה
- [ ] מלאי פיזי נשאר ללא שינוי עד אישור טעינה
- [ ] משטחים נבחרים לפי תאריך כניסה (ישן → חדש)

### עמלות
- [ ] עמלה מחושבת לפי סה"כ משקל במחזור
- [ ] טיירים עובדים נכון (15%, 17%, 20%)
- [ ] עמלת ראש צוות (5%) מחושבת נכון

### CRM
- [ ] סטטיסטיקות לקוח מתעדכנות אוטומטית
- [ ] חיפוש לקוח לפי טלפון עובד

### הרשאות
- [ ] מפיץ רואה רק את ההזמנות שלו
- [ ] אדמין רואה הכל
- [ ] ראש צוות יכול לאשר החזרות
