# מדריך לשימוש בקישור הזמנה ציבורי

## איך ליצור קישור להזמנת תמרים ללא התחברות

### שלב 1: מצא את ה-ID של המפיץ

יש לך כמה אפשרויות:

1. **דרך הממשק הניהול:**
   - התחבר כמנהל
   - לך לדף "מפיצים" או "משתמשים"
   - מצא את המפיץ שברצונך ליצור עבורו קישור
   - העתק את ה-ID שלו (UUID)

2. **דרך מסד הנתונים:**
   ```sql
   SELECT id, full_name, email FROM profiles WHERE role = 'distributor';
   ```

### שלב 2: צור את הקישור

הקישור הוא בפורמט הבא:
```
https://your-domain.com/order/[DISTRIBUTOR_ID]
```

**דוגמה:**
אם ה-ID של המפיץ הוא `123e4567-e89b-12d3-a456-426614174000`, הקישור יהיה:
```
https://your-domain.com/order/123e4567-e89b-12d3-a456-426614174000
```

### שלב 3: שתף את הקישור

אתה יכול לשתף את הקישור ב:
- קבוצת ווטסאפ של היישוב
- הודעת SMS
- אימייל
- פוסט בפייסבוק/אינסטגרם
- כל פלטפורמה אחרת

## מה הלקוח יראה?

1. **דף נחיתה להזמנה:**
   - רשימת כל המוצרים הזמינים
   - אפשרות להוסיף מוצרים לעגלה
   - בחירת כמות (בק"ג)

2. **דף תשלום:**
   - סיכום ההזמנה
   - טופס למילוי פרטי יצירת קשר (שם, טלפון, אימייל)
   - פרטי נקודת האיסוף

3. **דף אישור:**
   - אישור שההזמנה התקבלה
   - מספר הזמנה
   - פרטי ההזמנה

## תכונות

✅ **ללא התחברות** - הלקוח לא צריך ליצור חשבון  
✅ **פשוט ונוח** - ממשק ידידותי וקל לשימוש  
✅ **שמירת עגלה** - העגלה נשמרת גם אחרי סגירת הדף  
✅ **יצירת לקוח אוטומטית** - פרטי הלקוח נשמרים במערכת  
✅ **הזמנה מיידית** - ההזמנה נוצרת מיד במערכת  

## הערות חשובות

1. **תקופת הזמנה:** כל הזמנה נשמרת עם תקופת הזמנה של 30 דקות. אם הלקוח לא משלם בזמן, ההזמנה תתבטל אוטומטית.

2. **ניהול הזמנות:** כל ההזמנות מופיעות במערכת הניהול תחת המפיץ הרלוונטי.

3. **יצירת לקוחות:** המערכת יוצרת אוטומטית רשומת לקוח בטבלת `customers` אם הלקוח לא קיים, או מעדכנת את הפרטים אם הוא כבר קיים.

4. **תשלום:** כרגע ההזמנה נוצרת עם סטטוס "ממתין לתשלום". המפיץ צריך לאשר את התשלום ידנית דרך המערכת.

## דוגמה להודעת ווטסאפ

```
🌴 הזמנת תמרים טריים! 🌴

להזמנה ישירה ללא התחברות:
https://your-domain.com/order/[DISTRIBUTOR_ID]

✅ מבחר רחב של תמרים איכותיים
✅ מחירים מעולים
✅ איסוף מהיר מנקודת האיסוף

לשאלות: [מספר טלפון]
```

## פתרון בעיות

**הקישור לא עובד?**
- ודא שה-ID של המפיץ נכון
- ודא שהמפיץ קיים במערכת ופעיל
- בדוק שהאתר עובד (אם עדיין לא עשית דיפלוי, זה לא יעבוד)

**ההזמנה לא נוצרת?**
- בדוק את הלוגים של השרת
- ודא שיש מלאי זמין למוצרים
- ודא שהמפיץ קיים ופעיל
